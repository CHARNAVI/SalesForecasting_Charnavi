# 📊 End-to-End Sales Forecasting & Demand Intelligence System

## 🚀 Project Overview
This project is an end-to-end data science solution that analyzes retail sales data, forecasts future demand, detects anomalies, and segments products using machine learning techniques. It also includes an interactive Streamlit dashboard for business insights.

---

## 🎯 Objectives
- Clean and preprocess sales data
- Perform exploratory data analysis (EDA)
- Conduct time series decomposition
- Build forecasting models (SARIMA, Prophet, XGBoost)
- Compare model performance
- Detect anomalies in sales patterns
- Segment products using clustering
- Build interactive dashboard using Streamlit

---

## 📂 Dataset
- Superstore sales dataset (`train.csv`)
- Includes: Order Date, Sales, Profit, Category, Region, Product Name

---

## 🧠 Key Features
### 📈 Forecasting
- SARIMA
- Prophet
- XGBoost
- Model comparison using MAE, RMSE, MAPE

### 🚨 Anomaly Detection
- Isolation Forest
- Z-Score method

### 📦 Product Segmentation
- KMeans Clustering
- PCA Visualization

### 📊 Dashboard
- Sales Overview
- Forecast Explorer
- Anomaly Detection
- Product Segments
